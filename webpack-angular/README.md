# Webpack-angular

A simple Project Based on Angular

<code>node v8.11.0</code> &nbsp; <code>yarn v1.7.0</code> &nbsp; <code>OS Windows 10</code>

#### Build Setup

``` bash
# install dependencies
yarn install

# serve with hot reload at localhost:8080
yarn run dev

# run production
yarn run build

```
