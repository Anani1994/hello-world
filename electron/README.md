# Electron

Building a Electron desktop development environment

<code>node v8.7.0</code> &nbsp; <code>npm v5.4.0</code> &nbsp; <code>OS windows 10</code>

#### Build Setup

``` bash
# install dependencies
npm install

# run
npm run start

# build electron application for production
npm run package

```
