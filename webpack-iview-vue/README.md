# Webpack-iview-vue

<code>node v8.9.1</code> &nbsp; <code>yarn v1.7.0</code> &nbsp; <code>OS macOS</code>

#### Build Setup

``` bash
# install dependencies
yarn install

# serve with hot reload at localhost:8080
yarn run dev

# production
yarn run build

```
